## Схема БД (кратко)

### Таблицы
- **raw_materials** — сырьё и материалы (442 позиции). Поля: `id, article, name, unit, category_id`
- **material_categories** — категории сырья
- **material_prices** — цены на сырьё с историей. Поля: `id, material_id, price_per_unit, valid_from`
- **recipes** — рецепты чаёв. Поля: `id, article, name, description, output_quantity, output_unit, notes, is_active`
- **recipe_ingredients** — состав рецепта. Поля: `id, recipe_id, material_id, sub_recipe_id, quantity, unit`
- **products** — SKU (готовые продукты). Поля: `id, article, name, type_id, package_size, package_unit, recipe_id`
- **product_types** — типы упаковки
- **sku_recipe_components** — состав SKU (рецепт + упаковочные материалы). Поля: `id, product_id, recipe_id, material_id, quantity, unit`
- **price_lists** — прайслисты
- **price_list_items** — позиции прайслиста

### Views
- **recipe_cost** — рекурсивный расчёт себестоимости рецепта (руб/кг). Поля: `recipe_id, recipe_article, recipe_name, output_quantity, output_unit, total_cost, cost_per_kg`
- **sku_cost** — себестоимость SKU. Поля: `product_id, sku_article, product_name, package_size, package_unit, blend_cost, packaging_cost, total_sku_cost`
- **product_pricing** — итоговое ценообразование

## Правила ведения данных

- Единицы: только **кг** и **шт** (граммы запрещены)
- Артикулы вносятся **строго как указано** (могут содержать `/`)
- Рецепт может содержать другой рецепт как компонент (`sub_recipe_id`)
- Цены в рублях, с историей (`valid_from`)
- `output_quantity = 1.000`, `output_unit = 'кг'` для всех рецептов

### Правила категорий (type_id) для SKU

| type_id | Категория | Когда использовать |
|---|---|---|
| 28 | Черные чаи | Моносорта чёрного чая |
| 29 | Зеленые чаи | Моносорта зелёного чая (Сенча и др.) |
| 31 | Пуэры | Пуэры моносорта |
| 32 | Красные чаи | Китайские красные чаи (Дянь Хун и др.) |
| 33 | Травяные моносорта | Гречишный, ройбуш без добавок |
| 37 | Черный чай с добавками | Чёрный чай + натуральные добавки, без ароматизаторов |
| 38 | Черный чай с ароматами | Чёрный чай + ароматизаторы (коды 25XX в составе) |
| 39 | Моносорта | Зелёный чай моносорта (Сенча) |
| 41 | Зеленый чай с ароматами | Зелёный чай + ароматизаторы |
| 42 | Улуны моносорта | Те Гуань Инь, Да Хун Пао без добавок |
| 44 | Улуны с ароматами | Улун + ароматизаторы |
| 46 | Травяные напитки с ароматом | Ройбуш/гречишный + ароматизаторы |
| 48 | Фруктово-ягодные напитки с ароматами | Каркаде/фруктовая основа + ароматизаторы |

**Ароматизаторы** — сырьё с артикулами в диапазоне 2500–2599 (Земляника 2586, Вишня 2506 и т.д.)

### Форматы артикулов SKU
- **-ПА500** — пакет 500 гр
- **-ЗИП100** — зип-пакет (Дой Пак) 100 гр
- **-10** / **-10Б** — ПЭТ-банка
